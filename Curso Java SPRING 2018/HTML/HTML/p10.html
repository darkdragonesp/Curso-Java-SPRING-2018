Para crear un nuevo proyecto en Eclipse, seleccionar la opción del menú principal:

File > New > Java Project

A continuación seleccionar la opción de menú para crear una nueva clase:

New > Class

No olvidar marcar la opción public static void main en el apartado Which method stubs would you like to create?

Definimos dos variables int x=5, y=3 y la suma de ambas en int z=x+y

El código queda de la siguiente forma:

        public static void main(String[] args) {
		int x=5, y=3;
		int z = x + y;
		System.out.println("El resultado es: " + z);
	}

Y el resultado:

El resultado es: 8
